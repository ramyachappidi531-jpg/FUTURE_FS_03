Brew & Bliss is a modern, responsive café website designed to digitize customer experience by enabling users to browse a digital menu, add items to a cart, and book a table online.

📌 Project Overview

The goal of this project is to build a real-world business website for a café and solve common problems such as:

Manual order handling

Uncertainty in table reservations

Lack of digital menu and checkout system

This website provides a smooth and interactive experience for customers using only front-end technologies.
<hr>
✨ Features

✅ Fully responsive and modern UI design

✅ Dynamic menu generation using JavaScript

✅ Add to cart functionality

✅ Live cart total calculation

✅ Sliding cart sidebar

✅ Table reservation form

✅ Image based food & beverage menu

✅ Clean and user-friendly layout

🧩 How It Works

Menu items are dynamically rendered from a JavaScript array.

Users can add multiple items to the cart.

The cart sidebar displays selected items and total price.

Users can book a table by selecting table number and time.

On checkout, the system redirects to a UPI payment link.
